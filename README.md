# PongThing
A Pong type game I made to practice using SDL2. Covered in Chapter 1 of "Game Programming in C++" by Sanjay Madhav. With personal modifications by me.

Controls:<br>
W / Up arrow: Move up <br>
S / Down arrow: Move down
